# com.hungnt.database

Runtime database service (`DatabaseService`, `BaseDataTable`) and **Google Sheet** import (Editor).

## Prerequisites (consumer project)

- **Odin Inspector** (Sirenix) in the project — used by `DatabaseService`, `DatabaseConfigWindow`, and demo-style tables.
- **com.hungnt.core** (declared in this package’s `package.json` dependencies).
